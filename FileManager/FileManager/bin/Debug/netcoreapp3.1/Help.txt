
List of available commands:
			
 help					Shows list of available commands.
 choosedrive				Shows available drives and chooses the drive.
 showfiles				Shows available files in current directory.
 copyfile				Choose the file from current working directory, then moves it to the written directory if possible.!
 deletefile 				Choose the file from current working directory and then delete it if possible.
 movefile 	 			Choose the file from current working directory, then moves it to the written directory if possible.
 chdir 					Changes the current working directory.
 back					Moves to the previous directory.
 maketxtfile 				Makes a .txt file in the current working directory, then writes the user input into file with the chosen encoding.
 readtxtfile 				Writes .txt file's content with the chosen encoding in console if possible.
 currpath				Shows current working directory path.
 jumpto 				Switches current working directory to written path if possible.
 exit					Ends the program.
 concatenate				

 Words written in '' are variables in commands.
